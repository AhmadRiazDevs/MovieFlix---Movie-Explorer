# MovieFlix - Movie Explorer

A responsive movie discovery demo app built with HTML, CSS, and JavaScript. MovieFlix lets users search, filter, and explore trending movies using The Movie Database API.

## Features

- Search movies by title
- Filter by genre, release year, and sort order
- Trending movies carousel
- Responsive dark UI with polished cards
- Search history dropdown and recently viewed movie support

## Project Structure

- `index.html` — Main page layout
- `style.css` — App styling and responsive header layout
- `script.js` — API logic, search/filter handling, and UI rendering

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/AhmadRiazDevs/MovieFlix---Movie-Explorer.git
   ```
2. Open `index.html` in a browser.
3. Optionally, run a local server for best results (recommended):
   ```bash
   npx http-server .
   ```

## Usage

- Type a movie name into the search field.
- Use the genre, year, and sort filters to narrow results.
- Click `Clear All` to reset filters and search.

## Notes

- The app uses The Movie Database API, so it requires a valid API key in `script.js`.
- If you want to customize the app, update the CSS for styling and the JavaScript for additional filtering behavior.
